       recipes_carpentry.txt). Werkzeug-Tags MUESSEN das base:-Praefix nutzen
       (tags[base:hammer]/[base:saw]), sonst wird das Werkzeug nicht erkannt und
       der Craft committet nicht.

       TEST-MODUS (bewusst billig): jedes Rezept braucht nur 1 Baumstamm + Werkzeug
       und Woodwork 1, damit das Craften schnell durchgetestet werden kann. Das
       echte Material-/Skill-Balancing kommt danach - das sind reine Zahlen im
       Rezeptbaum ohne Code-Einfluss. */

    craftRecipe Holzwagen_WagenkastenT1
    {
        timedAction = Making,
        time = 120,
        Tags = AnySurfaceCraft;Carpentry,
        category = Carpentry,
        SkillRequired = Woodwork:1,
        xpAward = Woodwork:10,
        inputs
        {
            item 1 tags[base:hammer] mode:keep flags[MayDegradeVeryLight],
            item 1 [Base.Log] flags[Prop2],
        }
        outputs

            item 1 Base.Holzwagen_T1,
