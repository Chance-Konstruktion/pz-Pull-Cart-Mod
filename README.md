# Holzwagen (Pull Cart) für Project Zomboid

Ein **ziehbare** einachsiger Holzwagen für Build 42 – inspiriert von Valheim.

### Features

- **Ziehen mit beiden Händen** → realistisches Tempo (80% bei T1, 100% bei T2)
- **Zwei Stufen**:
  - T1: Einfacher Holzwagen (günstig)
  - T2: Verbesserte Version mit Speichenrädern (schneller)
- **Taschen-Slots** an den Seiten (T1: 2 Slots | T2: 4 Slots)
- **Fasswagen-Variante** – großes Flüssigkeitsfass (450 Einheiten) + 3 Taschen-Slots
- Vollständige Rezepte + Rückbau möglich
- Saubere, performante Lua-Logik

### Installation

1. Den Ordner `HolzwagenMod` in deinen Mods-Ordner kopieren:
   `C:\Users\DEINNAME\Zomboid\mods\HolzwagenMod`
2. Modelldateien (`holzwagen_t1.fbx`, `holzwagen_t2.fbx`, `holzwagen_fass.fbx`) nach  
   `HolzwagenMod/42/media/models_X/` legen
3. Im Spiel **Mods** aktivieren und ein neues Spiel starten

### Wichtige Hinweise

- Der Wagen belegt **beide Hände** beim Ziehen.
- Tempo wird dynamisch angepasst (abhängig von den verbauten Rädern).
- Fasswagen: Das Fass ist fest verbaut und kann mit Schlauch + Trichter befüllt/entleert werden.

### Rezepte (Carpentry)

- Holzwagen T1 & T2
- Rad-Upgrade T1 → T2
- Fasswagen-Umbau + Rückbau
